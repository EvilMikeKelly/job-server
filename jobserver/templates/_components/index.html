    Breadcrumbs

    Breadcrumb item

    Properties

    • active: [boolean] - true if the current page matches this breadcrumb item
    • location: [string] - shows a tooltip on hvoer displaying the text
    • title: [string] - the text to display for the breadcrumb
    • url: [string] - the anchor href for the breadcrumb
    {% verbatim breadcrumbs %}{# Title is special, if it is equal to "home" then the home icon will show #}
{% breadcrumb title="Home" url=home_url %}

{# URL will make the breadcrumb contain an anchor link #}
{% breadcrumb location="Organisation" title="Bennett Institute for Applied Data Science" url="#" %}

{# Active will make the breadcrumb not contain a link #}
{% breadcrumb location="Action" title="create_components" active=True %}
{% endverbatim breadcrumbs %}
